## Description
探索アルゴリズムのA* Saerch Algorithmを実装してみました。  
迷路などにおけるスタートからゴールまでの経路を探索します。

## Useage
マウスクリックやドラッグで障害物を設計した後
キーボードのｓで探索を開始します。  
探索後や障害物を再設計する場合にはキーボードでcを入力するとクリアできます。

## Environment
* Ubuntu16.04
* openFrameworks v0.10.0
* visualstudio code

## Originality
* 障害物を簡単に設計可能
* 探索途中の描画
